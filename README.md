# TRAIN-TICKET-PROJECT
PROGRAMMING LAMGUAGE USED:-C langguage 
Features
.View Train Details
  Displays the list of trains, including train number, name, source, destination, and ticket price.
.Book Ticket
  Allows users to book a ticket by entering passenger details and selecting a train.
.View All Bookings
  Displays all booked tickets from the storage file.
.Search Booking
  Retrieves passenger booking details using ticket number or name.
.Cancel Ticket
  Removes a booking from the records.
.File Handling Support
  Data (train details & bookings) is stored using files, so records persist after the program closes.
  
Technologies Used
.Programming Language: C
.Concepts Applied:
  Structures
  File Handling (read/write/update)
  Functions
  Menu-driven console interface

  Usage Instructions
Run the program.
Choose an option from the main menu:
1. View Trains
2. Book Ticket
3. View Bookings
4. Search Booking
5. Cancel Ticket
6. Exit
Follow the on-screen instructions for booking or managing tickets.
